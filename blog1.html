As embarrassing as it may sound, I am fairly new to HTML while others are pretty familiar to it.
Therefore, for my first blog post/learning experience, I wanted to explore CSS. I used these sites
to conduct my research about CSS.

    w3Schools
    MDN Web Docs

CSS stands for Cascading Style Sheets. CSS is extremely useful because it is what gives
webpages their interesting looks. Basically, HTML is given very basic styles by the
browser; these features include:
  • links are highlighted
  • understandable lists and tables
  • headings are larger than normal text
During the last two labs, we were introduced to CSS and its properties. Examples of our uses
with CSS include changing the color of text, creating boxes around code blocks, and adding
borders/rounded borders. I would say the cool thing about CSS is that its like its own class
within an HTML file.
