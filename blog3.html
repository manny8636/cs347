Blog Post #3

Continuing our theme of cool CSS properties, today we stumble across will-change. This property
alerts the browers which and what changes to expect on an element in the near future. These changes
are likely part of an animation - which makes sense because this property used to be named will-animate.
Look at these links to see how will-change is implemented:

    CSS-Tricks
    W3 Docs
The will-change property also gives browsers the ability to change an element's scroll position and contents. It is important to note that values must be comma-seperated; this property can have the following values:
    unset
    initial
    inherit
    will-change
    auto
    scroll-position
    contents
