Final Blog Post

Firstly, I believe that this class should have a prerequisite because there is
so much to learn and it would be helpful to come into the class knowing most of
basics. If I were teaching this class, I would have weekly quizzes on the readings
and limit the number of labs in the class to one per week; they would be graded more
for accuracy rather than completion. What helped me learn in this class was the textbook.
It was very manageable and contain a lot of useful information that was directly related to
our labs and projects.
